
# Intro

This project is inspired by [URL Snake](https://urlsnake.com/).

## Help

The build system for this project is CMake.
Setup build folders:
'''bash
cmake -B build
'''

Build project (same as 'make' when using Makefiles):
'''bash
cmake --build build
'''
